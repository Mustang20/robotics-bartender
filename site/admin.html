Double-click any text on the page to edit it. Click an image to change its file. Switch EN/ES to edit each language. Then press Save changes.

Run this through a local server

The editor needs the site to be served over http:// (not opened directly from disk) so it can read the page and your content file.

From this folder run, for example:
python -m http.server
then open http://localhost:8000/admin.html.

Or just edit content in Pages CMS after pushing to GitHub. See the README.
